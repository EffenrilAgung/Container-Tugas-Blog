Cara Membuat routing menggunakan EIGRP yang dimana saya menggunakan beberapa perangkat yaitu

  1. Komputer 3
  2. Router 3
  1. Persiapan Cisco
    • Langkah pertama sebelum melakukan pemetaan ip langkah pertama yang di persiapkan adalah mengumpulkan device 3 komputer dan 3 router seperti pada gambar berikut.
    • Setelah kita mengumpulkan device yang akan kita perlukan, melakukan penambahan perangkat berupa port pada router yang sudah di sediakan sebelum melakukan penambahan perangkat keras, perlu di perhatikan bahwa kita harus mematikan perangkat router terlebih dahulu sebelum melakukan penambahan perangkat keras seperti pada gambar terapkan cara tersebut ke 3 perangkat router lainnya.
    • Jangan lupa untuk menghidupkan kembali tombol power router tersebut
    • Selanjutnya lakukan pemetaan alamat ip pada device yang akan kita lakukan simulasi agar kita tidak kebingungan saat melakukan setting alamat ip
  2. Configurasi Alamat IP Pada Router
    • Router 1
      • Pada tab menu cli di router 1 ketik perintah sebagai berikut Router > enable Router#configure terminal Router# interface FastEthernet0/0 Router(config)#ip address 192.168.1. 254 255.255.255.0 Router(config-if)#no shutdown Router(config-if)# exit Router(config)# interface Serial9/0 Router(config-if)#ip address 10.10.10.1 255.0.0.0 Router(config-if)#clock rate 2000000 Router(config-if)# exit
    • Router 2
      • Router > enable Router#configure terminal Router# interface FastEthernet0/0 Router(config)#ip address ip address 192.168.20. 254 255.255.255.0 Router(config-if)#no shutdown Router(config-if)# exit Router(config)# interface Serial9/0 Router(config-if)#ip address 10.10.10.2 255.0.0.0 Router(config-if)#clock rate 2000000 Router(config-if)# exit Router(config)# interface Serial8/0 Router(config-if)#ip address 9.9.9.1 255.0.0.0 Router(config-if)#clock rate 2000000 Router(config-if)# exit
    • Router 3
      • Router > enable Router#configure terminal Router# interface FastEthernet0/0 Router(config)#ip address 192.168.30.254 255.255.255.0 Router(config-if)#no shutdown Router(config-if)# exit Router(config)# interface Serial9/0 Router(config-if)#ip address 9.9.9.2 255.0.0.0 Router(config-if)#clock rate 2000000 Router(config-if)# exit

  5. Mendaftarkan induk alamat ip ke router (Routing Information Protocol)
    • Pada tab menu Cli pada router masukkan perintah berikut dan terapkan ke semua router
      • Router > enable Router#configure terminal Router# router rip Router(config-if)#network 10.10.10.0 Router(config-if)#network 192.168.10.0 Router(config-if)#network 192.168.20.0 Router(config-if)#network 9.9.9.0 Router(config-if)#network 192.168.30.0
  6. Testing
    • Coba lakukan ping pada alamat ip router jika belum berhasil tunggu sebentar karena router mungkin sedang melakukan delay untuk beberapa saat, apabila ping berhasil coba untuk mengirimkan pesan dari komputer 1 ke komputer lainnya jika berhasil maka akan tampak seperti gambar di bawah
